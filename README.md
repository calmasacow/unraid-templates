# calmasacow Unraid templates

Unraid Community Applications templates for containers maintained by calmasacow.

## Templates

- `templates/torlink-plusplus.xml` — TorLink++: a Dockerized Torznab-compatible torrent search/download gateway built on a fork of the original Torlink project by bairon.

## TorLink++

Project repo:

https://github.com/calmasacow/TorLink-PlusPlus

Docker image:

```text
calmasacow/torlink-plusplus:latest
```

Template raw URL:

```text
https://raw.githubusercontent.com/calmasacow/unraid-templates/main/templates/torlink-plusplus.xml
```
